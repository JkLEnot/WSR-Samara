Для запуска интерфейсов требуется установить 'python3' и следующий список пакетов:
flask
web3

Настройка проекта:
  1. Открыть папку с нужным интерфейсом (CLI или GUI), открыть файл settings.py в текстовом редакторе
  2. По ключу 'Flask' в GUI заменить port и host на ваш
  3. По ключу 'Contract' в обоих интерфейсах заменить host и port на те, на которых запущена нода
  4. По ключам 'main', 'insurance' и 'dps' в 'Contract' заменить зачения 'address' и 'abi' каждого контракта
     в зависимости от название контракта ('Main', 'Insurance', 'DPS')
  5. Сохранить изменения

Запустить интерфейс можно двумя способами:
  1. Запустить bat файл нужного вам интерфейса (CLI, GUI) (GUI для лучшего отображения откройте в полный экран)
  3. Открыть консоль указав путь к директории с проектом,
     после чего прописать следующую команду - 'python CLI' или
     'python GUI' в зависимости от нужного вам интерфейса 

CLI:
  Для полного списка команд нужно ввести 'help'
  тег inline у команд значит что переменная будет введена с новой строки
